# ogc-sprint-2026-01

Demonstration of pygeoapi, Pub/Sub, OGC API - Publish-Subscribe, libheif and OGC API - Maps as part of the [2026 Joint OGC – OSGeo – ASF Code Sprint](https://github.com/opengeospatial/developer-events/wiki/2026-Joint-OGC-%E2%80%93-OSGeo-%E2%80%93-ASF-Code-Sprint).

This repository contains the following Docker Compose based demonstrations:

- `pygeoapi-heif-maps-demo`: pygeoapi custom OGC API - Maps provider for HEIF
- `pygeoapi-pubsub-demo`: pygeoapi implementation of OGC API - Publish-Subscribe
